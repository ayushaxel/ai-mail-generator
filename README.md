# AI Mail Generator 🚀

AI Mail Generator is a full-stack MERN application that generates professional emails instantly using AI.

## ✨ Features

* User Authentication (JWT)
* AI Email Generation using Groq AI
* Save & View Email History
* Responsive UI
* Copy to Clipboard

## 🛠️ Tech Stack

* React.js + Vite + Tailwind CSS
* Node.js + Express.js
* MongoDB
* Groq AI (Llama 3.3)
* JWT Authentication

## ⚙️ Installation

```bash
git clone https://github.com/Ayushaxel/ai-mail-generator.git
cd ai-mail-generator
```

### Backend

```bash
cd server
npm install
npm run dev
```

### Frontend

```bash
cd client
npm install
npm run dev
```
